Overview

API Reference

The NexusAPI REST API lets you integrate AI generation, user management, data processing, and webhooks into any application. All requests use HTTPS and return JSON.

Base URL: https://api.nexusapi.dev/v2

Authentication

Pass your API key in the Authorization header with every request.

Header 
Authorization: Bearer YOUR_API_KEY

Error Codes

CodeMeaningDescription
200OKRequest succeeded
400Bad RequestInvalid parameters
401UnauthorizedMissing or invalid API key
404Not FoundResource not found
429Too Many RequestsRate limit exceeded
500Server ErrorInternal server error

Core

GET /api/ping Returns server status

Health check endpoint. Returns current server status and version info.

Example Request
cURL 
curl -X GET https://api.nexusapi.dev/v2/ping \
  -H 'Authorization: Bearer YOUR_API_KEY'
Response 200 OK
JSON 
{
  "status": "online",
  "version": "2.4.0",
  "latency": 42,
  "region": "us-east-1"
}

AI Endpoints

POST /api/ai/generate Generate AI text response

Generates an AI-powered text response from a prompt using our hosted LLM.

Request Body
ParameterTypeRequiredDescription
promptstringrequiredThe input text to generate from
modelstringoptionalModel ID. Default: nexus-1
max_tokensintegeroptionalMax tokens to generate. Default: 512
temperaturefloatoptionalCreativity 0–1. Default: 0.7
Example Request
cURL 
curl -X POST https://api.nexusapi.dev/v2/ai/generate \
  -H "Authorization: Bearer YOUR_API_KEY" \
  -H "Content-Type: application/json" \
  -d '{"prompt": "hello", "max_tokens": 100}'
Response 200 OK
JSON 
{
  "response": "Hello! How can I help you today?",
  "model": "nexus-1",
  "tokens_used": 18,
  "latency_ms": 320
}
POST /api/ai/chat Multi-turn conversation

Send a conversation history and receive a contextual reply.

Request Body
ParameterTypeRequiredDescription
messagesarrayrequiredArray of {role, content} objects
systemstringoptionalSystem prompt to set context
streambooleanoptionalStream response via SSE. Default: false
Example Request
curl -X POST https://api.nexusapi.dev/v2/ai/chat \
  -H "Authorization: Bearer YOUR_API_KEY" \
  -H "Content-Type: application/json" \
  -d '{
    "messages": [
      {"role": "user", "content": "What is the capital of France?"}
    ],
    "system": "You are a helpful geography assistant."
  }'
Response
{
  "message": {
    "role": "assistant",
    "content": "The capital of France is Paris."
  },
  "tokens_used": 24
}

Users

GET /api/users List all users
Query Parameters
ParameterTypeRequiredDescription
pageintegeroptionalPage number. Default: 1
limitintegeroptionalResults per page. Default: 20, Max: 100
searchstringoptionalFilter by name or email
Response
{
  "data": [
    { "id": "usr_abc123", "name": "Jane Doe", "email": "jane@example.com", "created_at": "2025-01-01T00:00:00Z" }
  ],
  "pagination": { "page": 1, "limit": 20, "total": 142 }
}
POST /api/users Create a new user
Request Body
ParameterTypeRequiredDescription
namestringrequiredFull name
emailstringrequiredEmail address
rolestringoptionaluser or admin. Default: user
Example Request / Response
curl -X POST https://api.nexusapi.dev/v2/users \
  -d '{"name": "Jane Doe", "email": "jane@example.com"}'

// Response 201 Created:
{ "id": "usr_abc123", "name": "Jane Doe", "email": "jane@example.com", "role": "user" }
DELETE /api/users/:id Delete a user by ID
Path Parameters
ParameterTypeRequiredDescription
idstringrequiredUser ID (usr_…)
Response
{ "deleted": true, "id": "usr_abc123" }

Request Body
ParameterTypeRequiredDescription
